# Personal Portfolio Website

A personal portfolio website built using React.js to showcase my projects, technical skills, achievements, and development journey.

## Features

* Modern and responsive design
* Interactive and user-friendly interface
* Project showcase section
* Skills and technology overview
* Contact section
* Optimized for desktop and mobile devices

## Tech Stack

* React.js
* JavaScript
* HTML5
* CSS3

This portfolio was developed to create a professional online presence and showcase my frontend development skills. While building this project, I focused on responsive design, reusable components, clean UI, and smooth user experience across different devices.
